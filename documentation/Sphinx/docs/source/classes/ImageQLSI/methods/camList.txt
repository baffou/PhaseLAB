.. dropdown:: **camList**
    



    Displays the list of predefined camera names.

    .. code-block:: python

        >> MI.camList

        ans =

            5x1 cell array

            {'Sid4Element-Sona'}
            {'Sid4Element'     }
            {'Silios_mono'     }
            {'sC8-830'         }
            {'sC8-944'         }


    This list gives the possible values of the ``CGcam`` input in the Microscope constructor.
