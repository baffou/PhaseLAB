.. dropdown:: **ZernikeRemove**
    
    Removes the Zernike moments of the OPD image of an ImageQLSI object.

    **Synthax**
    
    .. include:: ../hr.txt

    .. code-block:: python

        obj.ZernikeRemove();
        obj.ZernikeRemove(n);
        obj.ZernikeRemove(n,m,r);
        obj2 = obj.ZernikeRemove(n,m,r);

    **Output arguments**




    .. code-block:: python

        IM.ZernikeRemove(n,m,r0);
        IM2 = IM.ZernikeRemove(n,m,r0);


    | ``n``, ``m`` are the orders of the Zernike moment.
    | ``r0`` is the radius of the area over which the Zerninke moment is calculated.

    If an output ``IM2`` is specified, IM is copied. If not, IM is modified.
